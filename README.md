# Fresh Guard
Fresh Guard is a simple and easy to use tool that helps you keep your pantry fresh by reminding you to use your food
before it goes bad.

It a mobile first web application that allows you to add items to your pantry, set expiration dates, and receive
notifications when items are about to expire.

# Installation
To install Fresh Guard, follow these steps:
1. Clone the repository:
   ```bash
   git clone git@github.com:marcelorodrigo/freshguard.git
    ```
2. Run the following command to install the dependencies:
   ```bash
   composer install
   npm install
   ```
